# Registro de Testes de Software (Em construção)

O Registro de Teste de Software oferece uma análise completa das funcionalidades testadas, identificação de erros encontrados, descreve ações corretivas tomadas e oferece uma avaliação abrangente da qualidade do software.

Abaixo, segue o relatório com as evidências dos testes de software realizados no sistema pela equipe, baseado em um plano de testes pré-definido para a aplicação Personal Cash.
